# AstraPort Core

AstraPort Core is the AI engine powering AstraPort. It analyzes Stellar wallet portfolios,
provides risk scoring, diversification strategies, and predictive insights.

## Repo Structure
- /docs        → Documentation for AI models
- /examples    → Sample scripts for portfolio analysis
- /src         → Core AI modules

## Modules
- Data Ingestion: Fetches wallet and market data
- Risk Scoring: ML models for volatility and exposure
- Diversification: Suggests asset allocation strategies
- Prediction: Forecasts market trends

## Getting Started
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the AI engine: `python main.py`

## Examples
See `/examples` for sample portfolio analysis scripts.
